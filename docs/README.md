## Contexto del producto

**SciBack School** es un sistema de gestión escolar para colegios privados peruanos construido sobre Odoo Community 17.0 + OpenEduCat. Modelo single-tenant: cada cliente recibe su propia instancia dedicada en AWS `us-east-2` (Ohio).

**Cliente alfa:** Escuela Cristiana Agua Viva (Lima, Perú).
